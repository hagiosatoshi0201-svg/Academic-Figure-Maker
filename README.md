# Academic-Figure-Maker
Academic Figure Maker Complete v5 is a privacy-first, browser-based tool for creating publication-ready scientific figures. Running 100% locally ensures your data stays secure.  Features:  17 chart types (Violin, Forest, Box, etc.)  High-res exports (PNG, SVG, PDF, PPTX)  Journal presets &amp; auto-stats  Simply open the HTML file to start!
